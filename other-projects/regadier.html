Regadier

Regadier is a pathfinding simulator and map editor developed using SDL2 in C. The simulation runs a pathfinding algorithm, Dijkstra or A*, in a grid. The algorithm can be configured to allow moving in diagonals. The editor also allows exporting and importing maps in PNG format. The grid is color coded using the following rules:

  • Blue: Start
  • Green: Finish
  • Red: Blocked path
  • Grayscale: Cost of the cell, from lowest (white) to highest (black).
